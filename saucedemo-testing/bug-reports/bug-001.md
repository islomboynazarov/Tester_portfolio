**Bug ID: BUG_SORT_001**

**Title:** Sorting dropdown is not clickable on Products page

**Environment**

Application: Sauce Demo

URL: https://www.saucedemo.com/

Browser: Brave 1.86.139

OS: Windows 10

**Preconditions**

User is logged in successfully

Products page is displayed

**Steps to Reproduce**

Navigate to the Products page

Locate the sorting dropdown at the top of the product list

Attempt to click ON the sorting dropdown icon

**Actual Result**

Sorting dropdown does not respond to click

Sorting options are not displayed

**Expected Result**

Sorting dropdown should be clickable

Sorting options should be displayed allowing the user to select a sorting method

**Severity**

Medium
(Core functionality is affected, but user can still browse products)

**Priority**

High
(Sorting is a key user experience feature and should work correctly)

**Status**

Open
