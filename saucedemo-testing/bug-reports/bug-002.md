**Bug Report**

**Bug ID: BUG_SORT_002**

**Title:** Sorting icon does not perform automatic sorting action when clicked

**Environment**

Application: Sauce Demo

URL: https://www.saucedemo.com/

Browser: Brave 1.86.139

OS: Windows 10

**Preconditions**

User is logged in successfully

Products page is displayed

**Steps to Reproduce**

1. Navigate to the Products page

2. Locate the sorting/filter icon (funnel icon) near the product list

3. Click on the sorting icon

**Actual Result**

Clicking the sorting icon triggers sorting method list

Product order remains unchanged

**Expected Result**

Clicking the sorting icon should automatically sort products based on a predefined default order (e.g., price or name)

Product list should update immediately after clicking the icon

**Severity**

Medium
(Sorting functionality is affected, but products are still accessible)

**Priority**

**Medium**
(User experience issue, not a complete functional blocker)

**Status**

Open
